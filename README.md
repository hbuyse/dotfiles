# DOTFILES

All my dotfiles used at work or at home

## Installation

First, download your age-key.txt file.
Then, you can copy this line of code:

```
sh -c "$(curl -fsLS chezmoi.io/getlb)" -- init --apply --source ${HOME}/.dotfiles hbuyse
```
